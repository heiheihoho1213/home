此方案的样式还可以通过 hook 动态获取为 token 使用。

优点：

  • 样式切换会更加顺畅
  • 便于组件封装与状态管理
  • 可配置性比较强，支持动态自定义样式

缺点：

  • 首屏加载会慢一些
  • 除了样式切换不会闪烁，其余缺点与动态 link 方案缺点一样
